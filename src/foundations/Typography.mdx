# Typography Tokens

Typography variables dictate consistent type scaling and legibility throughout the FMDQ application environment. Note: See the **Typography** under the Components section for React component documentation.

## Font Family
`--font-family-primary`: 'Clash Grotesk', sans-serif;

## Font Scale
Current established tokens in the root:
- `--font-size-sm`: 14px
- `--font-size-lg`: 16px
