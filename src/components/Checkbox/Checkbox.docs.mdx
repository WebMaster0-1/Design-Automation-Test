## Usage Guidelines
- Use Checkboxes when the user can select **any number of choices** (zero, one, or multiple).
- If the user must select **exactly one** choice, use the Radio component instead.
- Always provide a clear, concise `label`. Avoid using checkboxes without labels.
- When presenting a list of checkboxes, align them vertically for better scannability.
