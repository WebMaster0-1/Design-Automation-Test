## Usage Guidelines
- Always use **Helper Text** to guide users on expected input format, instead of relying solely on placeholders.
- **Error States** must be accompanied by an **Error Message** explaining what went wrong and how to fix it.
- Never use the `sm` and `lg` input sizes in the same form context; keep sizing consistent.

## Parity Note
This component inherits styles mapped to `tokens.css`:
- **Focus Rings**: Mapped to `--color-primary-default` and `--color-destructive-default`
- **Typography**: Clash Grotesk Semibold, matching `--font-size-*` variables.
- **Sizing**: 35px height for SM variant, 48px height for LG variant (perfect match with Button component).
