## Usage Guidelines
- **Always use Radio buttons for mutually exclusive options** where the user *must* select exactly one choice.
- If the user can select multiple options, use the **Checkbox** component instead.
- If you have an exceptionally long list of options (e.g., more than 6), consider using a **Select** component to save space.
- Radio buttons in a group must share the same `name` attribute so the browser knows they are mutually exclusive.
