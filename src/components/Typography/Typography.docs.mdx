# Typography

FMDQ UI provides a comprehensive typographic hierarchy matching the precise design specifications exactly as structured in the FMDQ Design System 2026. The entire system is built utilizing **DM Sans**.

## Display

Use the `Heading` component with `level="display-lg"` or `level="display-sm"` for exceptionally large promotional text.

<Canvas>
  <Story of={TypographyStories.Display} />
</Canvas>

## Headings

Use the `Heading` component for standard section titles. Pass `level={1}` through `level={6}` to match the corresponding HTML hierarchy.

<Canvas>
  <Story of={TypographyStories.Headings} />
</Canvas>

## Paragraphs

Use the `Text` component for paragraphs, descriptions, and general body text. Modify its size using `size="lg" | "md" | "sm" | "xs"`.

<Canvas>
  <Story of={TypographyStories.Paragraphs} />
</Canvas>

## Captions

Use the `Text` component with caption sizes (`caption-lg`, `caption-sm`, `caption-xs`) for very small supplementary text or metadata. Standardly formatted in all caps.

  <Story of={TypographyStories.Captions} />
</Canvas>
