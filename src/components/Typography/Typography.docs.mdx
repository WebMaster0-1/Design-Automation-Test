# Typography

FMDQ UI provides dedicated descriptive components for text representation: `Heading`, `Text`, and `Label`.

## Headings
Use the `Heading` component for section titles. Use the `level` prop to determine the HTML element (`h1` - `h6`).

  <Story of={TypographyStories.Headings} />
</Canvas>

## Text
Use the `Text` component for paragraphs, descriptions, and general body text. Modify its size using the `size` prop.

  <Story of={TypographyStories.Texts} />
