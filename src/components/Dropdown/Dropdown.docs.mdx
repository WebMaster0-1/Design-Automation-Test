# Dropdown

The `Dropdown` component provides a generic action menu or context menu triggered by user interaction. Use this for navigation, actions, or secondary settings instead of form selections.

## Default Usage

Wrap a `trigger` component (like a `Button` or an Icon) and provide an array of `items`.

<Canvas>
  <Story of={DropdownStories.Default} />
</Canvas>

## Alignment

You can use the `align` prop (`left` or `right`) to control the popup positioning relative to the trigger.

  <Story of={DropdownStories.RightAligned} />
