### Sizing and Length
Textareas can accept standard HTML layout attributes like `rows` and `cols` to dictate their initial size. You can also make them stretch to fit their container using the `fullWidth` prop.
<Canvas of={TextareaStories.FullWidth} />

## Usage Guidelines
- Provide Help Text (`helperText`) to communicate formatting rules or length constraints.
- **Error States** must be accompanied by an **Error Message** explaining what went wrong and how to fix it.
- Never use a Textarea for short, single-line data (like names or email addresses). Use the `Input` component instead.
