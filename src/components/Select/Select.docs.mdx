## Variants

### Sizing
Selects come in two standard sizes, exactly mimicking the `Input` and `Button` heights (SM=35px, LG=48px). Use the `size` prop to dictate this.

### Usage Context
You can provide `placeholder` text which renders as a disabled, hidden first option. Use `defaultValue` or `value` to pre-select an option from the `options` array. You can also disable specific options within the array using the `disabled` property on the `SelectOption` interface.

## Usage Guidelines
- Provide a clear, actionable `placeholder` (e.g., "Select a currency") to guide the user when no option is inherently the default.
- If you have fewer than 4 mutually exclusive options, consider using the **Radio** component instead.
- If the list of options exceeds 15-20 items, consider falling back to a searchable Select component (currently unavailable in this native implementation) or use an Autocomplete input.
