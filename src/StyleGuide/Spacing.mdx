# Spacing Tokens

Systematic spacing regulates rhythm, balance, and grouping within UI structures.

Use the established CSS variables rather than hardcoding pixel values to ensure holistic scalability.

## Base Units

| Token | Size | Example use Cases |
| :--- | :--- | :--- |
| `--spacing-8` | **8px** | Component internal padding, tight icon margins |
| `--spacing-gap-10` | **10px** | Flexbox gaps |
| `--spacing-12` | **12px** | Input padding, List item gaps |
| `--spacing-16` | **16px** | Standard component padding, Layout gutters |
| `--spacing-24` | **24px** | Section paddings, large component gaps |

## Radii

Corners are strictly governed by specific radii tokens:
- `--radius-8`: **8px**
- `--radius-16`: **16px**
- `--radius-24`: **24px**
