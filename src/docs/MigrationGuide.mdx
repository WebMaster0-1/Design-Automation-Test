# Migration Guide

This guide is designed to assist developers upgrading `@fmdq/ui` across breaking changes.

## Upgrading to v0.1.0 (Initial Beta)
There are no existing components to migrate from. For consumers switching from inline tailwind logic to the FMDQ Component system, please follow the [Getting Started](/docs/getting-started--docs) guidelines and replace existing HTML elements with `@fmdq/ui` components systematically.
