# Component Playground

The playground is a sandbox environment where developers can interactively test and combine components. 

## How to use the controls

1. Navigate to any component under the **Components** or **Layout** sidebar navigation.
2. Select the **Controls** tab at the bottom panel of the Storybook viewport.
3. Adjust attributes like `variant`, `disabled`, `size`, or typing custom strings. The component will instantly reflect these changes in live time.

> Want a fully integrated experience? Try assembling a prototype in your local machine by importing multiple layout components together.
