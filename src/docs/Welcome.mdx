# Welcome to FMDQ Design System

The FMDQ Design System is a standardized library of React components, design tokens, and functional guidelines tailored specifically for FMDQ digital markets platforms. Our primary aim is to ensure **consistency**, **scalability**, and **developer efficiency**.

## What is this?
This portal serves as the single source of truth for all designers and developers building user interfaces for FMDQ. It leverages the exact tokens derived from Figma while offering a stable, documented `@fmdq/ui` React component library.

## Core Principles
1. **Consistency First** — Components follow the exact FMDQ brand geometry, reducing cognitive load on users.
2. **Accessible by Default** — Built with WCAG standards to ensure optimal accessibility.
3. **Enterprise Grade** — Scalable layouts, modular atoms, and comprehensive documentation to match modern frontend architectures.

> Feel free to explore the **Getting Started** section to learn how to consume this library in your own application!
