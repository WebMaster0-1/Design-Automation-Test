# Getting Started

The `@fmdq/ui` component library is designed to get you building FMDQ interfaces as fast as possible.

## 1. Installation

Install the library using npm. Note that peer dependencies like `react` and `react-dom` are required.

<div className="code-block">
  npm install @fmdq/ui
</div>

## 2. Usage

Import the foundational CSS into your application's entry point (such as `main.tsx` or `_app.tsx`):

<div className="code-block">
  import "@fmdq/ui/style.css";
</div>

Now, you can import and use any component:

<div className="code-block">
{`import { Button } from "@fmdq/ui";

export default function MyFeature() {
  return <Button variant="primary">Execute Trade</Button>;
}`}
</div>
